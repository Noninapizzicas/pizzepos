# Estándares de Nombres

Este documento define las convenciones de nomenclatura utilizadas en todo el proyecto PizzePOS para garantizar consistencia y claridad.

## Principios Generales

1. **Idioma**: Español para todo el código, variables, comentarios y documentación.
2. **Consistencia**: Mantener el mismo estilo en todo el proyecto.
3. **Descriptivo**: Nombres que expliquen claramente su propósito.
4. **Conciso**: Lo suficientemente breve sin sacrificar claridad.
5. **Sin acentos ni caracteres especiales** en nombres de variables, funciones, clases, etc.
